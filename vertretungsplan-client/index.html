About this Project

Dieses Projekt soll einen Vertretungsplan simulieren und besteht aus einem separaten Frontend und Backend. Das Frontend ist mit Vite in HTML, CSS und JavaScript geschrieben und das Backend in Java mit dem Spring Boot Framework. Für Security wird Spring Security und JWT (JSON Web Token) als Token verwendet. Als Datenbank wird eine MySQL-Datenbank verwendet. Mehr Informationen über dieses Projekt auf GitHub
